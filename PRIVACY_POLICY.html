Copy It - 隐私政策

最后更新日期：2025年11月3日

核心原则：Copy It 扩展不收集、不存储、不传输任何用户数据。所有功能均在本地运行，您的隐私受到完全保护。

1. 信息收集

Copy It 扩展不收集任何个人信息，包括但不限于：

  • 不收集您的浏览历史
  • 不收集您访问的网站地址
  • 不收集您复制的文本内容
  • 不收集您的个人身份信息
  • 不收集任何设备信息或使用统计

2. 数据处理

2.1 复制的文本内容

当您使用 Copy It 复制文本时：

  • 选中的文本内容仅保存在当前标签页的内存中
  • 历史记录（最多 10 条）仅存储在本地内存，不写入硬盘
  • 关闭标签页后，所有数据自动清除
  • 扩展不会将任何内容上传到服务器或第三方服务

2.2 剪贴板访问

Copy It 使用浏览器的剪贴板 API 将您选中的文本写入系统剪贴板。此操作：

  • 仅在您主动点击复制按钮时触发
  • 完全在本地执行，不涉及网络传输
  • 遵循浏览器的安全策略和权限控制

3. 权限说明

Copy It 请求以下权限，所有权限仅用于实现核心功能：

3.1 offscreen（离屏文档）

  • 用途：在少数网站或受限环境中，当页面脚本阻止剪贴板写入时，作为兜底方案完成复制操作
  • 说明：Offscreen 文档仅在需要时按需创建，不收集、不传输任何数据，操作完成后立即销毁

3.2 host_permissions: <all_urls>（主机权限）

  • 用途：在任意网站上注入浮窗 UI 组件，以便您在访问任何网页时都能使用复制功能
  • 说明：扩展仅读取您选中的文本内容（通过 Selection API），不访问页面的其他数据、Cookie、表单或浏览历史

4. 数据存储

Copy It 扩展：

  • 不使用任何持久化存储（不使用 localStorage、chrome.storage 等）
  • 所有数据仅保存在当前标签页的内存中
  • 刷新页面或关闭标签页后，所有数据自动清除

5. 第三方服务

Copy It 扩展：

  • 不使用任何第三方分析服务（如 Google Analytics）
  • 不使用任何广告追踪服务
  • 不与任何第三方共享数据
  • 不包含任何外部脚本或资源加载

6. 儿童隐私

Copy It 扩展不针对儿童设计，也不会故意收集 13 岁以下儿童的个人信息。由于我们不收集任何用户数据，因此该扩展可以被任何年龄段的用户安全使用。

7. 安全性

虽然 Copy It 不收集任何数据，但我们仍然：

  • 使用 Shadow DOM 技术隔离扩展样式，避免与网页冲突
  • 遵循 Chrome Extension Manifest V3 的最佳安全实践
  • 最小化权限请求，仅申请必需的权限
  • 所有代码开源，可在 GitHub 上公开审查

8. 隐私政策更新

我们可能会不定期更新本隐私政策。任何重大变更将通过以下方式通知您：

  • 在 Chrome Web Store 的扩展描述页面公告
  • 在 GitHub 项目主页更新说明
  • 更新本页面顶部的"最后更新日期"

建议您定期查看本隐私政策以了解最新信息。

9. 用户权利

由于 Copy It 不收集任何用户数据，因此：

  • 无需担心数据泄露风险
  • 无需请求删除数据（因为没有数据被存储）
  • 无需申请数据导出（因为没有数据被收集）
  • 您可以随时卸载扩展，不会留下任何痕迹

11. 适用法律

本隐私政策受中华人民共和国法律管辖。我们承诺遵守《中华人民共和国个人信息保护法》及其他相关法律法规。

重要提示：Copy It 是一个完全本地化的工具，不涉及任何服务器或云服务。您的所有操作都在您的设备上进行，隐私安全得到最大程度的保护。
