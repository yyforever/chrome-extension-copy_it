
**详细描述**（必填，建议格式化）：

```markdown
Copy It 是一款强大的网页复制工具，帮助你轻松突破网站的复制限制。

🔓 核心功能
• 突破复制限制 - 在任何网站自由复制文本
• 浮动按钮 - 常驻可拖动的浮窗，选中文字即可复制
• 历史记录 - 自动保存最近 10 条复制记录
• 批量合并 - 支持多选历史并合并复制，自动换行
• 智能检测 - 选区 ≥2 字符自动激活

✨ 用户体验
• 无闪烁交互 - 流畅的 UI 更新
• 样式隔离 - 使用 Shadow DOM，不影响网页
• 即时反馈 - 复制成功/失败即时提示

🛠️ 使用方法
1. 在网页上选中文本（≥2 个字符）
2. 点击右上角高亮的浮窗即可复制
3. 点击浮窗打开菜单，查看历史记录
4. 勾选多条记录，点击"全部复制"批量合并

📋 复制限制
• 单条复制：≤1000 字符
• 超出限制：提供手动复制框
• 历史记录：最多 10 条（当前标签页内）

🔒 隐私保护
• 不收集任何用户数据
• 历史记录仅保存在本地内存
• 关闭标签页自动清空

⚠️ 注意事项
• 菜单中的"升级 Pro"功能目前仅为演示界面，无实际付费功能
• 部分使用特殊技术的网站可能仍无法复制

💻 技术
• Manifest V3
• Shadow DOM 样式隔离
• 多重复制兜底方案（Clipboard API → Offscreen Document → 手动）

如有问题或建议，欢迎访问项目主页反馈。
```

**类别**（必选）：
```
生产力工具 (Productivity)
```

**语言**：
```
中文（简体）
```

```
本扩展的唯一用途是：在用户选中网页文本时，显示一个可拖动浮窗；用户点击浮窗后将本次选中文本复制到剪贴板，并在菜单中管理最近 10 条复制记录以便合并复制。扩展不收集、不上传任何用户数据。
```

---

### 2️⃣ 权限用途说明（Permissions Justification）

**路径：隐私权 → 隐私权规范 → 权限用途说明**

#### offscreen 权限
```
在少数网站或受限环境中，页面脚本可能阻止剪贴板写入。扩展会在用户点击复制按钮后按需创建一个 Offscreen Document，在扩展隔离环境中完成剪贴板写入，避免站点脚本干扰。Offscreen 页面不收集、不传输任何数据，并在操作完成后立即销毁。
```

#### host_permissions: <all_urls>
```
用于在任意网站上注入浮窗 UI 组件，以便用户在访问任何网页时都能使用复制功能。扩展仅读取用户选中的文本内容（通过 Selection API），不访问页面的其他数据、Cookie、表单或浏览历史。
```
